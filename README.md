# Intro

Just the homebrew cask tap for [Keenotes Desktop](https://github.com/keevol/keenotes-desktop)

# How to use

## install

`brew tap keevol/keenotes && brew install --cask keenotes`

## upgrade

`brew update && brew upgrade --cask keenotes`
